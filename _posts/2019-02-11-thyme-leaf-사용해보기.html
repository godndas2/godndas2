Thymeleaf 사용해보기

- Thymeleaf ?

Thymeleaf는 Spring Boot의 view template engine으로서 가장 많이 사용하는 template engine이다.

변수

text (th:text)

변수 값을 태그의 텍스트로 표시할 때 th:text 속성과 ${...} 구문을 사용한다.

{% raw %} 
    
    /* Java */
    String username = "ReSpace";
    model.addAttribute("username", username);
{% endraw %} {% raw %} 
    

    <!-- Thymeleaf  -->
    <p th:text="${username}">name</p>
{% endraw %} {% raw %} 
    

    <!-- HTML 출력 결과 -->
    <p>ReSpace</p>
{% endraw %}

unescape text (th:utext)

변수의 값을 escape 처리하지 않고 표시할 때에는 th:utext 속성을 사용한다.

{% raw %} 
    

    <p th:utext="#{home.welcome}">Welcome to our grocery store!</p>
    This will output our message just like we wanted it:
    <p>Welcome to our <b>fantastic</b> grocery store!</p>
{% endraw %}

Text inlining(th:inline)

Script 태그에서 Controller로부터 전달된 변수 값으로 표시하려 할 경우 th:inline 속성을 사용하고 표시할 변수를 [[…]]로 묶는다.

{% raw %} 
    

    <!-- Thymeleaf -->
    <p th:inline="text">Hello, [[${session.user.name}]]!</p>
{% endraw %} {% raw %} 
    

    <!-- HTML 출력 결과 -->
    <p>Hello, ReSpace</p>
{% endraw %}

문자열 연결

+로 문자열과 변수를 연결할 수 있다.

{% raw %} 
    

    <!-- Thymeleaf -->
    <p th:text="'Hello, ' + ${username}"></p>
{% endraw %} {% raw %} 
    

    <!-- HTML 출력 결과 -->
    <p>Hello, ReSpace</p>
{% endraw %}

치환 하기

|${…}|를 사용하면 텍스트 내에 변수 값을 포함시킬 수 있다.

{% raw %} 
    

    <!-- Thymeleaf -->
    <p th:text="|Hello, ${username}|"></p>
{% endraw %} {% raw %} 
    

    <!-- HTML 출력 결과 -->
    <p>Hello, ReSpace</p>
{% endraw %}

기본값 설정하기

기본값을 정의하는 방법으로는 ?: 또는 #string 이라는 표현 도구 객체(Expression Utility Object)를 사용한다.
